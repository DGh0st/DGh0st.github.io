    iOS Runtime Modifications Projects

    These are extensions, for jailbroken devices, that I work on during my free time. The extensions fulfull numerous requests of the /r/jailbreak community and range from shifting UI elements for aesthetic purposes to completely changing a feature of the default iOS. The total download for the packages that have been published on a public repository (TheBigBoss) can be viewed at DGh0st's TheBigBoss stats

    UndoRotation


    GitHub
    Whenever you rotate your device, a button will be displayed on the screen allowing you to either undo rotation, lock orientation or both.
    Repo: BigBoss, DGh0st Beta Repo

    WhiteListSwitcherClose


    GitHub
    Create a whitelist of applications that will make applications irremovable, restart or launch when you try to close it via application switcher.
    Repo: BigBoss, DGh0st Beta Repo

    AlertClose


    GitHub
    Display an alert with option to close, relaunch, dismiss switcher and cancel when you swipe application up through application switcher.
    Repo: BigBoss, DGh0st Beta Repo

    AppSwitcherCurrentOrig


    GitHub
    Opens the app switcher to the current application instead of the last application. Official version is named AppSwitcherCurrent.
    Repo: BigBoss (iOS 8), BigBoss (iOS 9 & 10), DGh0st Beta Repo

    BadgeOverlay


    GitHub
    Make badges overlay on top of icons instead of being in the corner. Give badges a more sleek look by making them overlays.
    Repo: DGh0st Beta Repo

    PlayMafia

    PlayMafia is a multi-player online game based on the party game "Mafia." I designed and developed the communication system and assited in programing several roles for the gameplay. The website for the game was sadly taken down so there is no place to play it but the code for server, game and website is on GitHub.

    Hexagon Fort

    Hexagon Fort is a 2D single-player tower defense/offense game against an easy computer (AI). It is developed using Simple and Fast Multimedia Library (SFML). I worked on this project alone, to get a better understanding of SFML for future projects.
    Download by follwing the links: [zipped dropbox version] and [unzipped version]. Also the code for the game can be viewed on GitHub.
