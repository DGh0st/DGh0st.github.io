    Game A Week Challenges

    These are fun little small projects where my friend and I attempted to create a game in a week to understand new concepts/game engines.
    • Dungeon Of Ooze (Week 1)
    • The Gallant Gauntlet (Week 2 - 3)
    Dungeon of Ooze is a 3D single-player game which takes place inside a dungeon with monsters around every corner. It is developed using Unity.
    Download by follwing the links: [unzipped dropbox version]. Also the code for the game can be viewed on GitHub.
    The Gallant Gauntlet is a 2D multi-player game. It is developed using Simple and Fast Multimedia Library (SFML).
